# 語文我最棒

國小六年級國語智慧教學平台原型，提供教師備課、電子書教學、評量任務、學習紀錄與數位儀表板。

## 使用方式

直接開啟 `index.html` 即可使用。

快速體驗：

- 教師帳號：`teacher`
- 學生帳號：`student`
- 密碼：`best1234`

## GitHub Pages 部署

這是純靜態網站，可部署到 GitHub Pages：

1. 建立 GitHub repository。
2. 上傳 `index.html`、`styles.css`、`app.js`、`README.md`、`.nojekyll`。
3. 到 repository 的 Settings → Pages。
4. Source 選擇 `Deploy from a branch`。
5. Branch 選擇 `main` / root。
6. 儲存後等待 GitHub Pages 產生網址。
